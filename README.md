## Pentesting Report
1. (Required) XSS in YouTube URL Embeds
  - [ ] Summary: 
    - Vulnerability types: Stored XSS in WP Core
    - Tested in version: 4.2
    - Fixed in version: 4.2.13
  - [ ] GIF Walkthrough: <img src="pro7_1.gif" width="800">
